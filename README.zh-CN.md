## ✨ 功能特性

### 🎨 设计系统
- **三主题系统**：未来（未来科技）、赛博（赛博朋克）、商务（专业商务）
- **OKLCH 颜色空间**：感知均匀的颜色空间，支持 HEX 回退
- **语义化令牌**：所有组件一致的设计令牌
- **响应式设计**：移动优先的响应式设计原则
- **深色模式支持**：完整的深色模式令牌和主题切换

### 🛠️ 开发
- **TypeScript 支持**：完整的类型定义和类型安全的令牌访问
- **组件库**：50+ 可重用的 UI 组件
- **自动化构建**：Style Dictionary 用于令牌转换
- **单一事实来源**：`design/tokens.json` 作为事实来源
- **Figma 集成**：Figma 与代码之间的双向同步

### 🧪 质量与测试
- **全面测试**：1000+ 测试用例，80%+ 覆盖率
- **视觉回归**：Chromatic 自动化视觉测试

- **无障碍测试**：Jest-axe 无障碍验证
- **性能监控**：Lighthouse CI 性能跟踪

### ⚡ 性能
- **包大小优化**：gzip 压缩后 <200KB
- **Tree Shaking**：支持 ESM 的 tree-shaking
- **代码分割**：React.lazy 自动代码分割
- **性能预算**：强制执行的性能预算
- **Web Vitals**：核心 Web Vitals 监控

### 🌍 国际化
- **双语支持**：中文（zh-CN）和英文（en-US）
- **语言环境验证**：自动化语言环境验证

### 🔒 安全
- **XSS 防护**：DOMPurify 集成防止 XSS

## 🚀 快速开始

### 安装

```bash
# 使用 npm
npm install yyc3-design-system

# 使用 pnpm
pnpm add yyc3-design-system

# 使用 yarn
yarn add yyc3-design-system
```

### 基础用法

```tsx
import { Button, Card, Input } from 'yyc3-design-system';
import { ThemeProvider } from 'yyc3-design-system/theme';

function App() {
  return (
    <ThemeProvider theme="future">
      <Card>
        <h1>欢迎使用 YYC³ 设计系统</h1>
        <Button variant="primary">开始使用</Button>
        <Input placeholder="请输入您的姓名" />
      </Card>
    </ThemeProvider>
  );
}

export default App;
```

### 主题切换

```tsx
import { useTheme } from 'yyc3-design-system/theme';

function ThemeSwitcher() {
  const { theme, setTheme } = useTheme();

  return (
    <div>
      <button onClick={() => setTheme('future')}>未来</button>
      <button onClick={() => setTheme('cyber')}>赛博</button>
      <button onClick={() => setTheme('business')}>商务</button>
      <p>当前主题：{theme}</p>
    </div>
  );
}
```

## 🎭 主题

### 未来主题（未来科技）
- **颜色**：霓虹蓝、紫色、青色渐变
- **排版**：现代无衬线字体
- **风格**：未来、科技导向
- **用例**：技术、创新、初创公司

### 赛博主题（赛博朋克）
- **颜色**：粉色、红色、霓虹绿强调色
- **排版**：大胆、前卫字体
- **风格**：赛博朋克、未来、大胆
- **用例**：游戏、娱乐、创意

### 商务主题（专业）
- **颜色**：蓝色、灰色、白色调色板
- **排版**：专业衬线/无衬线字体
- **风格**：企业、专业、清洁
- **用例**：企业、商务、公司

### 主题切换

```tsx
import { useTheme } from 'yyc3-design-system/theme';

function ThemeSwitcher() {
  const { theme, setTheme } = useTheme();

  return (
    <div className="theme-switcher">
      <button
        onClick={() => setTheme('future')}
        className={theme === 'future' ? 'active' : ''}
      >
        未来
      </button>
      <button
        onClick={() => setTheme('cyber')}
        className={theme === 'cyber' ? 'active' : ''}
      >
        赛博
      </button>
      <button
        onClick={() => setTheme('business')}
        className={theme === 'business' ? 'active' : ''}
      >
        商务
      </button>
    </div>
  );
}
```
