使用指南

遵循这些指南可以确保项目的一致性和可维护性。

1

安装

通过npm安装YYC³ Design System

npm install yyc3-design-system
2

导入

在项目中导入需要的组件

import { Button } from 'yyc3-design-system';
3

使用

在React组件中使用导入的组件

<Button>点击我</Button>
4

主题

使用ThemeProvider包裹应用以启用主题切换

<ThemeProvider>...</ThemeProvider>
