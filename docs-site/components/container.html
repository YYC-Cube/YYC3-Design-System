Container 组件

Container 组件是一个响应式容器组件，支持多种最大宽度配置。用于居中内容并控制内容宽度。

基础示例

这是一个默认的 Container 组件，最大宽度为 1024px。

API 属性

属性 类型 默认值 说明
maxWidth 'sm' | 'md' | 'lg' | 'xl' | '2xl' | 'full' | 'none' 'lg' 最大宽度
children ReactNode - 容器内容

最大宽度选项

sm - 640px

小容器，适用于移动设备

md - 768px

中等容器，适用于平板设备

lg - 1024px

大容器，适用于桌面设备

xl - 1280px

超大容器，适用于大屏幕

2xl - 1536px

特大容器，适用于超大屏幕

full - 100%

全宽容器

使用示例

import { Container } from 'yyc3-design-system';

function App() {
  return (
    <Container maxWidth="lg">
      <h1>页面标题</h1>
      <p>页面内容</p>
    </Container>
  );
}

最佳实践

  • 使用 maxWidth 属性来控制内容宽度
  • Container 会自动居中内容
  • 使用 Container 来包裹主要内容区域
  • 根据内容类型选择合适的最大宽度
  • Container 会自动添加左右内边距
