Grid 组件

Grid 组件是一个响应式网格布局组件，支持自定义列数和间距。用于创建灵活的网格布局。

API 属性

属性 类型 默认值 说明
cols number 1 网格列数
gap number | string 1 网格间距（rem单位或自定义值）
children ReactNode - 网格内容

使用示例

import { Grid } from 'yyc3-design-system';

function App() {
  return (
    <Grid cols={3} gap={2}>
      <div>项目 1</div>
      <div>项目 2</div>
      <div>项目 3</div>
      <div>项目 4</div>
      <div>项目 5</div>
      <div>项目 6</div>
    </Grid>
  );
}

最佳实践

  • 使用 cols 属性来控制网格列数
  • 使用 gap 属性来控制网格间距
  • 网格会自动适应屏幕大小
  • 使用 Grid 组件来创建响应式布局
  • 保持网格内容的一致性
