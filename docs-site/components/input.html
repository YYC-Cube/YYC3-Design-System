Input 组件

Input 组件是一个表单输入框，支持多种类型、验证和样式定制。用于收集用户输入。

基础示例

API 属性

属性 类型 默认值 说明
type 'text' | 'email' | 'password' | 'number' | 'tel' | 'url' 'text' 输入框类型
label string - 标签文本
placeholder string - 占位符文本
disabled boolean false 是否禁用
required boolean false 是否必填
value string - 输入值（受控组件）
onChange (value: string) => void - 值变化回调

输入类型

Text

普通文本输入

Email

邮箱格式验证

Password

密码输入，内容隐藏

Number

数字输入

使用示例

import { Input } from 'yyc3-design-system';

function App() {
  const [username, setUsername] = useState('');
  const [email, setEmail] = useState('');

  return (
    <div>
      <Input
        type="text"
        label="用户名"
        placeholder="请输入用户名"
        value={username}
        onChange={setUsername}
      />
      <Input
        type="email"
        label="邮箱"
        placeholder="example@yyc3.com"
        value={email}
        onChange={setEmail}
      />
    </div>
  );
}

最佳实践

  • 始终提供清晰的 label 属性
  • 使用 placeholder 提供输入提示
  • 为必填字段设置 required 属性
  • 使用受控组件模式管理输入值
  • 根据输入类型选择合适的 type 属性
  • 禁用的输入框应该有明确的视觉反馈
