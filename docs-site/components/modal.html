Modal 组件

Modal 组件是一个弹出式对话框，支持遮罩层、键盘导航和点击外部关闭。用于显示重要信息或表单。

基础示例

API 属性

属性 类型 默认值 说明
isOpen boolean false 是否显示模态框
onClose () => void - 关闭回调函数
children ReactNode - 模态框内容
className string - 自定义类名

特性

  • ✓ 点击遮罩层关闭
  • ✓ ESC 键关闭
  • ✓ 阻止背景滚动
  • ✓ 响应式设计
  • ✓ 无障碍支持（ARIA）

使用示例

import { useState } from 'react';
import { Modal } from 'yyc3-design-system';

function App() {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <div>
      <button onClick={() => setIsOpen(true)}>
        打开模态框
      </button>
      
      <Modal isOpen={isOpen} onClose={() => setIsOpen(false)}>
        <h2>模态框标题</h2>
        <p>这是模态框的内容。</p>
        <button onClick={() => setIsOpen(false)}>
          关闭
        </button>
      </Modal>
    </div>
  );
}

最佳实践

  • 模态框应该有明确的关闭方式
  • 使用 onClose 回调来管理模态框状态
  • 模态框内容应该简洁明了
  • 避免嵌套模态框
  • 使用 ESC 键关闭以提高可访问性
  • 模态框打开时应该阻止背景滚动
