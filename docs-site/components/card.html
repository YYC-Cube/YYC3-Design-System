Card 组件

Card 组件是一个容器组件，用于内容分组和展示。支持Header、Title、Content等子组件。

这是卡片的内容区域。可以放置文本、图片、按钮等各种内容。

组件 说明

使用示例

import { Card, CardHeader, CardTitle, CardContent } from 'yyc3-design-system';

function App() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>卡片标题</CardTitle>
      </CardHeader>
      <CardContent>
        这是卡片的内容区域。
      </CardContent>
    </Card>
  );
}

最佳实践

  • 使用 CardHeader 和 CardTitle 来组织卡片标题
  • 使用 CardContent 来包裹卡片内容
  • 卡片内容应该简洁明了，避免过度复杂
  • 使用卡片来分组相关内容
  • 在响应式布局中，卡片可以自动适应屏幕大小
