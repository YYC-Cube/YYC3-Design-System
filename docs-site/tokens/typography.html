字体

字体系统定义了字体家族、字号、行高等排版规范，确保界面的一致性和可读性。

字体家族

Sans Serif
Inter, -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif
Monospace
SF Mono, Monaco, 'Cascadia Code', 'Roboto Mono', monospace

字号

H1 - 2.5rem (40px)
标题一
H2 - 2rem (32px)
标题二
H3 - 1.5rem (24px)
标题三
Body - 1rem (16px)
正文内容
Small - 0.875rem (14px)
小号文本

字重

Regular (400)
普通文本
Medium (500)
中等字重
Semibold (600)
半粗体
Bold (700)
粗体

行高

Tight (1.2)
紧凑行高，适用于标题
Normal (1.6)
正常行高，适用于正文
Relaxed (1.8)
宽松行高，适用于长文本

使用指南

  • 使用 Sans Serif 字体作为主要字体
  • 使用 Monospace 字体显示代码和数据
  • 标题使用较大的字号和较重的字重
  • 正文使用适中的行高，提高可读性
  • 保持字体层级的一致性
